# Part B — Vivado Port of VGA Controller

## Overview

Part B ports an existing VGA 640×480 controller from Intel Quartus (Cyclone V) to AMD Vivado (Artix-7). The goal is to demonstrate **cross-toolchain proficiency** — the VHDL logic stays identical; only the vendor IP and constraint format change.

---


## `src/vga_controller.vhd`

### Role

Generates standard VGA timing signals for 640×480 @ 60 Hz. This file is **imported unchanged** from the Quartus project. No edits to the core logic are needed — porting only affects the vendor IP wrappers.

### VGA 640×480 @ 60 Hz Timing

```
Pixel clock: 25.175 MHz (≈ 25 MHz from Clocking Wizard)

Horizontal timing (pixels):
  Active:        640
  Front porch:   16
  Sync pulse:    96  (active LOW)
  Back porch:    48
  Total:         800

Vertical timing (lines):
  Active:        480
  Front porch:   10
  Sync pulse:    2   (active LOW)
  Back porch:    33
  Total:         525
```

### Entity Interface

```vhdl
entity vga_controller is
    port (
        clk_25mhz  : in  std_logic;         -- Pixel clock (25 MHz)
        rst        : in  std_logic;

        -- VGA sync outputs
        hsync      : out std_logic;          -- Horizontal sync (active low)
        vsync      : out std_logic;          -- Vertical sync (active low)

        -- Current pixel coordinates (for pixel generator)
        pixel_x    : out std_logic_vector(9 downto 0);   -- 0–639
        pixel_y    : out std_logic_vector(9 downto 0);   -- 0–479
        video_on   : out std_logic;          -- '1' during active display area

        -- RGB output (to DAC or direct pins)
        red        : out std_logic_vector(3 downto 0);
        green      : out std_logic_vector(3 downto 0);
        blue       : out std_logic_vector(3 downto 0)
    );
end entity vga_controller;
```

### Internal Counters

```vhdl
-- Horizontal counter: 0 to 799
signal h_cnt : unsigned(9 downto 0);

-- Vertical counter: 0 to 524
signal v_cnt : unsigned(9 downto 0);

-- Horizontal counter process
process(clk_25mhz)
begin
    if rising_edge(clk_25mhz) then
        if rst = '1' or h_cnt = 799 then
            h_cnt <= (others => '0');
        else
            h_cnt <= h_cnt + 1;
        end if;
    end if;
end process;

-- Vertical counter (increments at end of each line)
process(clk_25mhz)
begin
    if rising_edge(clk_25mhz) then
        if rst = '1' then
            v_cnt <= (others => '0');
        elsif h_cnt = 799 then
            if v_cnt = 524 then
                v_cnt <= (others => '0');
            else
                v_cnt <= v_cnt + 1;
            end if;
        end if;
    end if;
end process;

-- Sync pulses
hsync <= '0' when h_cnt >= 656 and h_cnt < 752 else '1';  -- 640+16=656, 656+96=752
vsync <= '0' when v_cnt >= 490 and v_cnt < 492 else '1';  -- 480+10=490, 490+2=492

-- Active display area
video_on <= '1' when h_cnt < 640 and v_cnt < 480 else '0';

pixel_x <= std_logic_vector(h_cnt(9 downto 0));
pixel_y <= std_logic_vector(v_cnt(9 downto 0));
```

## `src/top.vhd` (Vivado Top-Level)

### What Changes vs. Quartus

| Component | Quartus Original | Vivado Port |
|-----------|-----------------|-------------|
| PLL | `ALTPLL` megafunction | Clocking Wizard IP (`clk_wiz_0`) |
| ROM | `altsyncram` megafunction | Block Memory Generator IP (`blk_mem_gen_0`) |
| Simulation | ModelSim | xsim (Vivado built-in) |
| Constraints | `.sdc` (Synopsys Design Constraints) | `.xdc` (Xilinx Design Constraints) |

### Entity

```vhdl
entity top is
    port (
        clk_100mhz : in  std_logic;   -- Artix-7 board: 100 MHz oscillator
        reset      : in  std_logic;   -- Active-high (BTN0 on Basys3)

        -- VGA connector
        hsync      : out std_logic;
        vsync      : out std_logic;
        vga_r      : out std_logic_vector(3 downto 0);
        vga_g      : out std_logic_vector(3 downto 0);
        vga_b      : out std_logic_vector(3 downto 0)
    );
end entity top;
```

### Clocking Wizard Instantiation

Generate via Vivado IP Catalog: **Tools → Create IP → Clocking Wizard**.
- Input: 100 MHz
- Output CLK_OUT1: 25 MHz (for pixel clock)
- Board: Artix-7 (`xc7a35tcpg236-1`)

```vhdl
u_clk : entity work.clk_wiz_0       -- Generated by IP Catalog
    port map (
        clk_in1  => clk_100mhz,
        clk_out1 => clk_25mhz,
        reset    => reset,
        locked   => pll_locked
    );
```

### Block Memory Generator Instantiation

Generate via Vivado IP Catalog: **Tools → Create IP → Block Memory Generator**.
- Memory type: Single Port ROM
- Width: 12 bits (for 4-bit RGB × 3 channels)
- Depth: matches image size
- Load `.coe` initialization file (convert `.mif` → `.coe` using the script below)

```vhdl
u_rom : entity work.blk_mem_gen_0   -- Generated by IP Catalog
    port map (
        clka  => clk_25mhz,
        addra => rom_addr,
        douta => rom_data
    );
```

## `constraints/artix7.xdc`

### Role

Replaces the Quartus `.sdc` file. Defines:
1. Clock constraint (timing analysis).
2. Pin assignments (physical I/O).
3. I/O standards.

### Implementation

```tcl
# constraints/artix7.xdc
# Target: Artix-7 xc7a35tcpg236-1 (e.g. Basys3 board)

# ── Clock ──────────────────────────────────────────────────────────────────
# 100 MHz input clock on pin W5 (Basys3)
create_clock -period 10.000 -name clk_100mhz [get_ports clk_100mhz]
set_property PACKAGE_PIN W5 [get_ports clk_100mhz]
set_property IOSTANDARD LVCMOS33 [get_ports clk_100mhz]

# ── Reset ──────────────────────────────────────────────────────────────────
set_property PACKAGE_PIN U18 [get_ports reset]
set_property IOSTANDARD LVCMOS33 [get_ports reset]

# ── VGA Connector (Basys3 Pmod JB/JC or dedicated VGA) ───────────────────
# Adjust pin assignments to your actual board schematic

# HSYNC
set_property PACKAGE_PIN P19 [get_ports hsync]
set_property IOSTANDARD LVCMOS33 [get_ports hsync]

# VSYNC
set_property PACKAGE_PIN R19 [get_ports vsync]
set_property IOSTANDARD LVCMOS33 [get_ports vsync]

# Red channel (4 bits)
set_property PACKAGE_PIN N19 [get_ports {vga_r[0]}]
set_property PACKAGE_PIN J19 [get_ports {vga_r[1]}]
set_property PACKAGE_PIN H19 [get_ports {vga_r[2]}]
set_property PACKAGE_PIN G19 [get_ports {vga_r[3]}]
set_property IOSTANDARD LVCMOS33 [get_ports {vga_r[*]}]

# Green channel
set_property PACKAGE_PIN F19 [get_ports {vga_g[0]}]
set_property PACKAGE_PIN E19 [get_ports {vga_g[1]}]
set_property PACKAGE_PIN E20 [get_ports {vga_g[2]}]
set_property PACKAGE_PIN D20 [get_ports {vga_g[3]}]
set_property IOSTANDARD LVCMOS33 [get_ports {vga_g[*]}]

# Blue channel
set_property PACKAGE_PIN C20 [get_ports {vga_b[0]}]
set_property PACKAGE_PIN B20 [get_ports {vga_b[1]}]
set_property PACKAGE_PIN A20 [get_ports {vga_b[2]}]
set_property PACKAGE_PIN B19 [get_ports {vga_b[3]}]
set_property IOSTANDARD LVCMOS33 [get_ports {vga_b[*]}]
```

> These are **placeholder** pin assignments. Adjust to your actual board. For simulation-only (no board), dummy pin assignments are fine — Vivado's xsim does not need real pins.

## `.mif` to `.coe` Conversion Script

Vivado's Block Memory Generator uses Xilinx `.coe` format instead of Intel `.mif`:

```python
#!/usr/bin/env python3
# mif2coe.py — Convert Intel MIF to Xilinx COE

import sys, re

def mif2coe(mif_path, coe_path):
    with open(mif_path) as f:
        lines = f.readlines()

    words = []
    in_content = False
    for line in lines:
        line = line.strip()
        if line.upper() == 'CONTENT BEGIN':
            in_content = True
            continue
        if line.upper() == 'END;':
            break
        if in_content and ':' in line:
            # Format: "ADDR : DATA;"  or  "[ADDR1..ADDR2] : DATA;"
            match = re.match(r'\[?[\w\.]+\]?\s*:\s*([\dA-Fa-f]+);', line)
            if match:
                words.append(match.group(1))

    with open(coe_path, 'w') as f:
        f.write("memory_initialization_radix=16;\n")
        f.write("memory_initialization_vector=\n")
        f.write(",\n".join(words))
        f.write(";\n")

if __name__ == '__main__':
    mif2coe(sys.argv[1], sys.argv[2])
```

## `PORTING_NOTES.md`

Document every difference you encounter. Minimum entries:

| Category | Quartus | Vivado | Notes |
|----------|---------|--------|-------|
| Clock IP | ALTPLL, wizard generates `.vhd` | Clocking Wizard, generates `.xci` | Parameter names differ |
| ROM IP | `altsyncram` | Block Memory Generator | Init file format: `.mif` → `.coe` |
| Constraints | `.sdc` with Tcl `create_clock` | `.xdc` with `set_property` | Different syntax for same function |
| Simulation | ModelSim `.do` scripts | xsim `.tcl` scripts | Both support TCL but different API |
| Synthesis time | Quartus: ~2 min | Vivado: ~3 min (on small designs) | — |
| Timing closure | Quartus: Fmax in Timing Analyzer | Vivado: WNS in Implementation Report | — |
